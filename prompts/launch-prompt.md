# Product Launch Prompt

## Objective

Create launch-ready marketing content for a product.

## Inputs

Product Name:
{{product_name}}

Category:
{{category}}

Target Audience:
{{audience}}

Problem:
{{problem}}

Solution:
{{solution}}

Key Features:
- {{feature_1}}
- {{feature_2}}
- {{feature_3}}

Tone:
{{tone}}

Platform:
{{platform}}

## Instructions

1. Understand the product.
2. Identify the target audience.
3. Focus on benefits over features.
4. Match the writing style to the selected platform.
5. Keep the content clear, concise, and engaging.
6. End with a strong call to action.

## Supported Platforms

- X (Twitter)
- Product Hunt
- Blog
- Press Release
- Email
- LinkedIn
- Landing Page
- Documentation

## Output

Return only the requested content.
