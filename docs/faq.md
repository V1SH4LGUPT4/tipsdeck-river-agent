# Frequently Asked Questions

## What is Tipsdeck River Agent?

Tipsdeck River Agent is an open-source AI skill repository designed to help developers, creators, and Web3 communities automate common workflows using reusable prompts, templates, documentation, and GitHub best practices.

---

## Who should use this repository?

- Developers
- Open-source maintainers
- AI builders
- Startup founders
- Web3 contributors
- Community managers
- Product launch teams

---

## What problems does it solve?

The repository reduces repetitive work by providing reusable documentation, prompt libraries, launch templates, release notes, and workflow examples.

---

## Does it require coding knowledge?

Basic Git and GitHub knowledge is helpful, but many templates and prompts can be used without writing code.

---

## Can I customize the prompts?

Yes.

Every prompt is written in Markdown and can be modified to match your project, writing style, or workflow.

---

## What platforms are supported?

- GitHub
- ProductClank
- X (Twitter)
- Blogs
- Documentation sites
- Developer portals

---

## Can I contribute?

Yes.

Community contributions are welcome through pull requests, issue reports, documentation improvements, prompt libraries, and new templates.

---

## Is this repository AI compatible?

Yes.

The repository is organized for AI agents using structured Markdown documentation, reusable prompts, examples, templates, and SKILL.md.

---
