# Contributing Guide

## Welcome

Thank you for your interest in contributing to Tipsdeck River Agent.

Our goal is to build a high-quality collection of reusable AI skills, prompts, templates, and documentation for developers and Web3 communities.

---

# Ways to Contribute

You can contribute by:

- Improving documentation
- Adding prompt libraries
- Creating templates
- Fixing issues
- Reporting bugs
- Improving examples
- Reviewing pull requests
- Sharing feature ideas

---

# Repository Standards

Please follow these guidelines:

- Write clear Markdown.
- Keep prompts reusable.
- Add meaningful commit messages.
- Avoid duplicate content.
- Test documentation before submitting.

---

# Pull Requests

Before submitting a pull request:

- Check existing issues.
- Keep changes focused.
- Update documentation if required.
- Add examples whenever possible.

---

# Documentation

Documentation should be:

- Easy to read
- Beginner friendly
- AI compatible
- Modular
- Well organized

---

# Prompt Guidelines

Every prompt should include:

- Purpose
- Inputs
- Expected Output
- Example Usage
- Notes

---

# Templates

Templates should be reusable.

Avoid project-specific hardcoded information.

---

# Community

We welcome:

- Developers
- AI builders
- Web3 contributors
- Open-source maintainers
- Community managers
